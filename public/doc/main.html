L'ensemble des objets a proximité

Body

Media type: application/json

Type: object

Properties
  • pokomons: required (array of Pokomon)

    Items: Pokomon

    • _id: required (string)
    • name: required (string)
    • number: required (integer)

      Le numéro dans le pokodex

    • picture: required (string)

      Une URL avec un png qui représente le pokomon

    • level: required (integer)
    • attaque: required (integer)
    • defense: required (integer)
    • vitesse: required (integer)
    • xp: required (integer)
    • captured: required (boolean)

      indique si le Pokomon est déjà capturé. Il fallait être plus rapide !

    • coordinates: required (array of number)

      coordinates[0] longitude, coordinates[1] latitude

    • parent_number: (string)

      Le pokomon vers lequel il peut evoluer

  • pokostops: required (array of Pokostop)

    Items: Pokostop

    • _id: required (string)
    • name: required (string)
    • coordinates: required (array of number)

      coordinates[0] longitude, coordinates[1] latitude

    • picture: required (string)
    • timer: (integer)

      Timer durant lequel on ne peut pas récupérer des Items
